# 公文格式主题

## 主题说明

这是一个为公文写作设计的Typora主题，符合《党政机关公文格式》(GB/T 9704-2012)标准。

## 标题映射

- `# 标题` → 文档标题（方正小标宋简体，二号，居中）
- `## 标题` → 一级标题（黑体，三号，左对齐）
- `### 标题` → 二级标题（楷体，三号，左对齐）
- `#### 标题` → 三级标题（仿宋，三号，左对齐）
- `##### 标题` → 四级标题（仿宋，四号，左对齐）

## 安装方法

1. 复制整个 `official-document` 文件夹到Typora主题目录
2. 在Typora中选择 **主题** → **official-document**
3. 重启Typora生效

## 特性

- 符合公文标准的字体设置
- 正确的页边距设置
- 打印优化
- 深色模式支持
- 响应式设计

## 字体要求

- 方正小标宋简体
- 黑体
- 楷体_GB2312
- 仿宋_GB2312

如果系统缺少这些字体，会自动降级到系统默认字体。

## 配合使用

配合 `convert_doc.bat` 脚本使用，可直接导出为符合标准的Word文档。
