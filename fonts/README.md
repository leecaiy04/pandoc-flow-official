# 字体要求和使用指南

## 📋 必需字体列表

根据《党政机关公文格式》(GB/T 9704-2012)标准，需要安装以下字体：

### 主要字体
| 字体名称 | 用途 | 字号要求 | 备注 |
|----------|------|----------|------|
| 方正小标宋简体 | 文档标题 (#) | 二号(22pt) | **必需**，公文正式标题 |
| 黑体 | 一级标题 (##) | 三号(16pt) | **必需** |
| 楷体_GB2312 | 二级标题 (###) | 三号(16pt) | **必需** |
| 仿宋_GB2312 | 正文、三级标题 | 三号(16pt) | **必需** |

### 备用字体
| 字体名称 | 用途 | 说明 |
|----------|------|------|
| Microsoft YaHei | 备用中文字体 | 当主要字体不可用时降级 |
| SimHei | 备用黑体 | 当黑体不可用时 |
| KaiTi | 备用楷体 | 当楷体_GB2312不可用时 |
| FangSong | 备用仿宋 | 当仿宋_GB2312不可用时 |

## 🔍 字体检查方法

### Windows系统检查
```cmd
# 检查方正小标宋
dir C:\Windows\Fonts\*方正小标宋*

# 检查黑体
dir C:\Windows\Fonts\*黑体*

# 检查楷体
dir C:\Windows\Fonts\*楷体*

# 检查仿宋
dir C:\Windows\Fonts\*仿宋*
```

### 注册表检查
```cmd
reg query "HKLM\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Fonts"
```

## 💾 字体安装方法

### 方法一：自动安装脚本
```batch
@echo off
echo 正在安装公文所需字体...

REM 方正小标宋简体（需要提供字体文件）
if exist "fonts\方正小标宋简体.ttf" (
    copy "fonts\方正小标宋简体.ttf" "%WINDIR%\Fonts\"
    reg add "HKLM\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Fonts" /v "方正小标宋简体" /t REG_SZ /d "方正小标宋简体.ttf" /f
    echo ✓ 方正小标宋简体安装完成
)

REM 黑体（Windows自带，无需安装）
echo ✓ 黑体已安装

REM 楷体（Windows自带，无需安装）
echo ✓ 楷体已安装

REM 仿宋（Windows自带，无需安装）
echo ✓ 仿宋已安装

echo 字体安装完成！
pause
```

### 方法二：手动安装
1. 右键字体文件 → "为所有用户安装"
2. 或拖拽字体文件到 `C:\Windows\Fonts\` 目录
3. 重启相关应用程序

## ⚠️ 常见问题

### 问题1：方正小标宋简体未安装
**症状：** 文档标题显示为宋体
**解决：** 
- 从官方渠道下载字体
- 联系IT部门获取字体安装包
- 使用其他标题字体作为替代

### 问题2：字体名称不匹配
**症状：** Word中找不到指定字体
**解决：** 
- 检查字体是否正确安装
- 确认字体名称是否完全匹配
- 使用字体管理器查看实际字体名称

### 问题3：权限不足
**症状：** 无法安装字体
**解决：** 
- 以管理员身份运行安装脚本
- 联系系统管理员

## 🔄 字体降级策略

当理想字体不可用时，系统将自动降级：

```
方正小标宋简体 → Microsoft YaHei → 宋体
黑体 → SimHei → Microsoft YaHei
楷体_GB2312 → KaiTi → Microsoft YaHei
仿宋_GB2312 → FangSong → SimSun
```

## 📁 字体文件存储

- **系统字体目录：** `C:\Windows\Fonts\`
- **用户字体目录：** `%APPDATA%\Microsoft\Windows\Fonts\`
- **项目字体目录：** `fonts/` (如果包含自定义字体)

## 📞 技术支持

如遇到字体相关技术问题：
1. 检查系统更新
2. 清除字体缓存
3. 重新安装字体
4. 重启系统
