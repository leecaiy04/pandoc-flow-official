# 公文标题级别映射规范（更新版）

## 标题级别映射关系

| Markdown 语法 | Word样式 | 字体 | 字号 | 样式 | 对齐方式 | 说明 |
|---------------|----------|------|------|------|----------|------|
| `# 标题` | 文档标题 | 方正小标宋简体 | 二号(22pt) | 加粗 | 居中 | 公文的正式标题 |
| `## 标题` | 一级标题 | 黑体 | 三号(16pt) | 加粗 | 左对齐 | 主要章节标题 |
| `### 标题` | 二级标题 | 楷体_GB2312 | 三号(16pt) | 加粗 | 左对齐 | 次级章节标题 |
| `#### 标题` | 三级标题 | 仿宋_GB2312 | 三号(16pt) | 加粗 | 左对齐 | 三级章节标题 |
| `##### 标题` | 四级标题 | 仿宋_GB2312 | 四号(14pt) | 加粗 | 左对齐 | 四级章节标题 |
| 普通文本 | 正文 | 仿宋_GB2312 | 三号(16pt) | 常规 | 首行缩进2字符 | 正文内容 |

## Word模板样式设置步骤

### 1. 文档标题样式（对应 # ）
- 在Word中：开始→样式→标题1
- 右键→修改→设置：
  - 字体：方正小标宋简体
  - 字号：二号(22pt)
  - 加粗：是
  - 对齐：居中

### 2. 一级标题样式（对应 ## ）
- 选择：标题2样式
- 设置：
  - 字体：黑体
  - 字号：三号(16pt)
  - 加粗：是
  - 对齐：左对齐

### 3. 二级标题样式（对应 ### ）
- 选择：标题3样式
- 设置：
  - 字体：楷体_GB2312
  - 字号：三号(16pt)
  - 加粗：是
  - 对齐：左对齐

### 4. 三级标题样式（对应 #### ）
- 选择：标题4样式
- 设置：
  - 字体：仿宋_GB2312
  - 字号：三号(16pt)
  - 加粗：是
  - 对齐：左对齐

### 5. 四级标题样式（对应 ##### ）
- 选择：标题5样式
- 设置：
  - 字体：仿宋_GB2312
  - 字号：四号(14pt)
  - 加粗：是
  - 对齐：左对齐

### 6. 正文样式
- 选择：正文样式
- 设置：
  - 字体：仿宋_GB2312
  - 字号：三号(16pt)
  - 加粗：否
  - 段落：首行缩进2字符，1.5倍行距

```markdown

```

## 转换脚本更新

### 新的Lua过滤器（pandoc_title_mapping.lua）

```lua
-- 公文标题映射过滤器
function Header (header)
  local level = header.level
  
  if level == 1 then
    -- # → 文档标题：方正小标宋简体，二号，加粗，居中
    return pandoc.Div(
      pandoc.Para{pandoc.Span(header.content, pandoc.Attr("", {}, {["font-family"] = "方正小标宋简体", ["font-size"] = "22pt", ["font-weight"] = "bold", ["text-align"] = "center"}))},
      pandoc.Attr("", {"document-title"}, {})
    )
  elseif level == 2 then
    -- ## → 一级标题：黑体，三号，加粗，左对齐
    return pandoc.Div(
      pandoc.Para{pandoc.Span(header.content, pandoc.Attr("", {}, {["font-family"] = "黑体", ["font-size"] = "16pt", ["font-weight"] = "bold", ["text-align"] = "left"}))},
      pandoc.Attr("", {"level-1-heading"}, {})
    )
  elseif level == 3 then
    -- ### → 二级标题：楷体_GB2312，三号，加粗，左对齐
    return pandoc.Div(
      pandoc.Para{pandoc.Span(header.content, pandoc.Attr("", {}, {["font-family"] = "楷体_GB2312", ["font-size"] = "16pt", ["font-weight"] = "bold", ["text-align"] = "left"}))},
      pandoc.Attr("", {"level-2-heading"}, {})
    )
  elseif level == 4 then
    -- #### → 三级标题：仿宋_GB2312，三号，加粗，左对齐
    return pandoc.Div(
      pandoc.Para{pandoc.Span(header.content, pandoc.Attr("", {}, {["font-family"] = "仿宋_GB2312", ["font-size"] = "16pt", ["font-weight"] = "bold", ["text-align"] = "left"}))},
      pandoc.Attr("", {"level-3-heading"}, {})
    )
  elseif level == 5 then
    -- ##### → 四级标题：仿宋_GB2312，四号，加粗，左对齐
    return pandoc.Div(
      pandoc.Para{pandoc.Span(header.content, pandoc.Attr("", {}, {["font-family"] = "仿宋_GB2312", ["font-size"] = "14pt", ["font-weight"] = "bold", ["text-align"] = "left"}))},
      pandoc.Attr("", {"level-4-heading"}, {})
    )
  end
  
  return header
end

function Para (para)
  -- 正文：仿宋_GB2312，三号，首行缩进2字符
  return pandoc.Div(
    para,
    pandoc.Attr("", {"body-text"}, {})
  )
end
```

## 更新的转换命令

```bash
# 使用更新映射的转换
pandoc input.md --reference-doc=公文模板.docx ^
    --lua-filter=pandoc_title_mapping.lua ^
    --variable CJKmainfont="方正小标宋简体" ^
    -o output.docx

# 或者使用批处理脚本
convert_doc_updated.bat input.md output.docx
```

## Word宏设置（可选）

按Alt+F11打开VBA编辑器，插入以下代码：

```vba
Sub UpdateOfficialDocStylesUpdated()
    ' 文档标题样式（对应 # ）
    With ActiveDocument.Styles(wdStyleHeading1).Font
        .NameAscii = "方正小标宋简体"
        .NameFarEast = "方正小标宋简体"
        .Size = 22
        .Bold = True
    End With
    ActiveDocument.Styles(wdStyleHeading1).ParagraphFormat.Alignment = wdAlignParagraphCenter
    
    ' 一级标题样式（对应 ## ）
    With ActiveDocument.Styles(wdStyleHeading2).Font
        .NameAscii = "黑体"
        .NameFarEast = "黑体"
        .Size = 16
        .Bold = True
    End With
    ActiveDocument.Styles(wdStyleHeading2).ParagraphFormat.Alignment = wdAlignParagraphLeft
    
    ' 二级标题样式（对应 ### ）
    With ActiveDocument.Styles(wdStyleHeading3).Font
        .NameAscii = "楷体_GB2312"
        .NameFarEast = "楷体_GB2312"
        .Size = 16
        .Bold = True
    End With
    ActiveDocument.Styles(wdStyleHeading3).ParagraphFormat.Alignment = wdAlignParagraphLeft
    
    ' 三级标题样式（对应 #### ）
    With ActiveDocument.Styles(wdStyleHeading4).Font
        .NameAscii = "仿宋_GB2312"
        .NameFarEast = "仿宋_GB2312"
        .Size = 16
        .Bold = True
    End With
    ActiveDocument.Styles(wdStyleHeading4).ParagraphFormat.Alignment = wdAlignParagraphLeft
    
    ' 四级标题样式（对应 ##### ）
    With ActiveDocument.Styles(wdStyleHeading5).Font
        .NameAscii = "仿宋_GB2312"
        .NameFarEast = "仿宋_GB2312"
        .Size = 14
        .Bold = True
    End With
    ActiveDocument.Styles(wdStyleHeading5).ParagraphFormat.Alignment = wdAlignParagraphLeft
    
    ' 正文样式
    With ActiveDocument.Styles(wdStyleNormal).Font
        .NameAscii = "仿宋_GB2312"
        .NameFarEast = "仿宋_GB2312"
        .Size = 16
        .Bold = False
    End With
    With ActiveDocument.Styles(wdStyleNormal).ParagraphFormat
        .CharacterUnitFirstLineIndent = 2
        .LineSpacingRule = wdLineSpace1pt5
        .Alignment = wdAlignParagraphLeft
    End With
    
    MsgBox "样式更新完成！#对应文档标题，##对应一级标题"
End Sub
```
