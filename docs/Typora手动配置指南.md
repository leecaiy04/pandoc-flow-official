# Typora 手动配置指南

由于编码问题，建议手动配置Typora。请按照以下步骤操作：

## 🎯 步骤1：安装主题

### 复制主题文件
1. 打开文件资源管理器
2. 导航到：`D:\Code\Template\pandoc\typora-config\typora-theme\`
3. 复制整个 `official-document` 文件夹
4. 粘贴到：`%APPDATA%\Typora\themes\`

**快速访问方式：**
- 按 `Win + R`，输入 `%APPDATA%\Typora\themes\` 并回车
- 如果themes文件夹不存在，请创建它

## 🎯 步骤2：配置Typora

### 应用主题
1. 打开Typora
2. 点击菜单栏的 **主题**
3. 选择 **official-document**
4. 如果看不到该选项，重启Typora

### 配置导出
1. 在Typora中点击 **文件** → **偏好设置**
2. 选择 **导出** 标签页
3. 在 **自定义导出** 部分点击 **添加自定义导出格式**

填写以下信息：
```
格式名称：公文格式Word
命令：convert_doc.bat
参数："${currentFilePath}" "${currentFilePath}.docx"
工作目录：D:\Code\Template\pandoc
```

### 设置快捷键（可选）
1. 切换到 **快捷键** 标签页
2. 找到 "公文格式 Word" 命令
3. 点击设置快捷键（推荐：Ctrl+Shift+D）

## 🎯 步骤3：验证配置

### 测试主题
1. 创建新文档
2. 输入以下内容：
```markdown
# 测试文档标题
## 一级标题测试
### 二级标题测试
#### 三级标题测试

这是正文内容，应该显示为仿宋字体。
```
3. 检查预览效果：
   - 标题是否居中且字体正确
   - 正文是否为仿宋字体

### 测试导出
1. 使用配置的导出功能
2. 检查生成的Word文档格式是否正确

## 🎯 步骤4：字体检查

确保以下字体已安装：
- ✅ 方正小标宋简体（文档标题）
- ✅ 黑体（一级标题）
- ✅ 楷体_GB2312（二级标题）
- ✅ 仿宋_GB2312（正文和三级标题）

## 🚨 常见问题

### 问题1：主题不生效
**解决方法：**
1. 确认主题文件复制到正确位置
2. 重启Typora
3. 检查主题是否在列表中

### 问题2：导出失败
**解决方法：**
1. 检查pandoc是否安装：在命令行输入 `pandoc --version`
2. 检查脚本路径是否正确
3. 确认工作目录设置正确

### 问题3：字体显示错误
**解决方法：**
1. 安装缺失的字体
2. 在Word中手动修改模板样式
3. 确认字体名称匹配

## 💡 快速验证脚本

创建一个测试文件 `test.md`：
```markdown
# 关于召开工作会议的通知

## 一、会议背景
为全面总结工作成果。

## 二、会议安排

### （一）时间地点
- 时间：2024年1月15日
- 地点：会议室

### （二）参会人员
1. 领导班子
2. 各部门负责人

## 三、会议议程

### （一）工作总结

#### 1. 上午议程
签到入场

#### 2. 下午议程
分组讨论

## 四、工作要求

各部门需按时完成相关工作。
```

然后运行：
```bash
scripts\convert_doc.bat test.md output.docx
```

检查生成的Word文档格式是否正确。

1. 确认所有路径正确
2. 检查文件权限
3. 重新启动Typora和系统
4. 查看Typora的错误日志
