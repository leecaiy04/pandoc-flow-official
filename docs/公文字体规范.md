# 公文字体规范和调整指南

## 标准公文字体要求

根据《党政机关公文格式》(GB/T 9704-2012)标准：

### 字体规范

| Markdown语法 | Word样式 | 字体 | 字号 | 样式 | 对齐方式 | 说明 |
|---------------|----------|------|------|------|----------|------|
| `# 标题` | 文档标题 | 方正小标宋简体 | 二号(22pt) | 加粗 | 居中 | 公文正式标题 |
| `## 标题` | 一级标题 | 黑体 | 三号(16pt) | 加粗 | 左对齐 | 主要章节标题 |
| `### 标题` | 二级标题 | 楷体_GB2312 | 三号(16pt) | 加粗 | 左对齐 | 次级章节标题 |
| `#### 标题` | 三级标题 | 仿宋_GB2312 | 三号(16pt) | 加粗 | 左对齐 | 三级章节标题 |
| `##### 标题` | 四级标题 | 仿宋_GB2312 | 四号(14pt) | 加粗 | 左对齐 | 四级章节标题 |
| 普通文本 | 正文 | 仿宋_GB2312 | 三号(16pt) | 常规 | 首行缩进2字符 | 正文内容 |

## Word模板修改步骤

### 方法一：手动修改模板

1. **打开 official-template.docx**
2. **修改文档标题样式（对应 # ）**：
   - 进入"开始"→"样式"面板
   - 找到"标题1"样式，右键→"修改"
   - 设置字体：方正小标宋简体
   - 设置字号：二号(22pt)
   - 设置加粗：是
   - 设置对齐：居中
   - 点击"确定"

4. **修改一级标题样式（对应 ## ）**：
   - 找到"标题2"样式，右键→"修改"
   - 设置字体：黑体
   - 设置字号：三号(16pt)
   - 设置加粗：是
   - 设置对齐：左对齐

5. **修改二级标题样式（对应 ### ）**：
   - 找到"标题3"样式，右键→"修改"
   - 设置字体：楷体_GB2312
   - 设置字号：三号(16pt)
   - 设置加粗：是
   - 设置对齐：左对齐

6. **修改三级标题样式（对应 #### ）**：
   - 找到"标题4"样式，右键→"修改"
   - 设置字体：仿宋_GB2312
   - 设置字号：三号(16pt)
   - 设置加粗：是
   - 设置对齐：左对齐

7. **修改四级标题样式（对应 ##### ）**：
   - 找到"标题5"样式，右键→"修改"
   - 设置字体：仿宋_GB2312
   - 设置字号：四号(14pt)
   - 设置加粗：是
   - 设置对齐：左对齐

6. **修改正文样式**：
   - 找到"正文"样式，右键→"修改"
   - 设置字体：仿宋_GB2312
   - 设置字号：三号(16pt)
   - 设置段落：首行缩进2字符
   - 设置行距：1.5倍行距

7. **保存模板**

### 方法二：使用Word宏批量修改

按Alt+F11打开VBA编辑器，插入以下代码：

```vba
Sub UpdateOfficialDocStyles()
    ' 修改公文标题样式
    With ActiveDocument.Styles(wdStyleHeading1).Font
        .NameAscii = "方正小标宋简体"
        .NameFarEast = "方正小标宋简体"
        .Size = 22
        .Bold = True
    End With
    ActiveDocument.Styles(wdStyleHeading1).ParagraphFormat.Alignment = wdAlignParagraphCenter
    
    ' 修改一级标题样式
    With ActiveDocument.Styles(wdStyleHeading2).Font
        .NameAscii = "黑体"
        .NameFarEast = "黑体"
        .Size = 16
        .Bold = True
    End With
    ActiveDocument.Styles(wdStyleHeading2).ParagraphFormat.Alignment = wdAlignParagraphLeft
    
    ' 修改二级标题样式
    With ActiveDocument.Styles(wdStyleHeading3).Font
        .NameAscii = "楷体_GB2312"
        .NameFarEast = "楷体_GB2312"
        .Size = 16
        .Bold = True
    End With
    ActiveDocument.Styles(wdStyleHeading3).ParagraphFormat.Alignment = wdAlignParagraphLeft
    
    ' 修改三级标题样式
    With ActiveDocument.Styles(wdStyleHeading4).Font
        .NameAscii = "仿宋_GB2312"
        .NameFarEast = "仿宋_GB2312"
        .Size = 16
        .Bold = True
    End With
    ActiveDocument.Styles(wdStyleHeading4).ParagraphFormat.Alignment = wdAlignParagraphLeft
    
    ' 修改正文样式
    With ActiveDocument.Styles(wdStyleNormal).Font
        .NameAscii = "仿宋_GB2312"
        .NameFarEast = "仿宋_GB2312"
        .Size = 16
        .Bold = False
    End With
    With ActiveDocument.Styles(wdStyleNormal).ParagraphFormat
        .CharacterUnitFirstLineIndent = 2
        .LineSpacingRule = wdLineSpace1pt5
        .Alignment = wdAlignParagraphLeft
    End With
    
    MsgBox "样式修改完成！"
End Sub
```

## 转换脚本使用

### 更新的转换命令

```bash
# 使用带字体规范的转换脚本
scripts\convert_doc.bat input.md output.docx

# 或者直接使用pandoc（推荐显式指定 defaults 和模板）
pandoc input.md -d templates/pandoc-defaults.yaml ^
    --reference-doc=templates/official-template.docx ^
    -o output.docx
```

### 验证字体安装

在转换前，请确保系统中已安装以下字体：

1. **检查字体是否安装**：
   ```cmd
   # 在命令行中查看字体
   dir C:\Windows\Fonts\*方正小标宋*
   dir C:\Windows\Fonts\*黑体*
   dir C:\Windows\Fonts\*楷体*
   dir C:\Windows\Fonts\*仿宋*
   ```

2. **如果字体未安装**：
   - 从官方渠道下载方正小标宋简体
   - Windows系统自带：黑体、楷体、仿宋

## Pandoc字体配置选项

### 创建自定义pandoc.lua过滤器

```lua
-- 字体设置过滤器
function Header (header)
  local level = header.level
  
  if level == 1 then
    -- 公文标题：方正小标宋简体，二号，加粗，居中
    return pandoc.Div(
      pandoc.Para{pandoc.Span(header.content, pandoc.Attr("", {}, {["font-family"] = "方正小标宋简体", ["font-size"] = "22pt", ["font-weight"] = "bold", ["text-align"] = "center"}))},
      pandoc.Attr("", {"official-title"}, {})
    )
  elseif level == 2 then
    -- 一级标题：黑体，三号，加粗，左对齐
    return pandoc.Div(
      pandoc.Para{pandoc.Span(header.content, pandoc.Attr("", {}, {["font-family"] = "黑体", ["font-size"] = "16pt", ["font-weight"] = "bold", ["text-align"] = "left"}))},
      pandoc.Attr("", {"level-1-heading"}, {})
    )
  elseif level == 3 then
    -- 二级标题：楷体_GB2312，三号，加粗，左对齐
    return pandoc.Div(
      pandoc.Para{pandoc.Span(header.content, pandoc.Attr("", {}, {["font-family"] = "楷体_GB2312", ["font-size"] = "16pt", ["font-weight"] = "bold", ["text-align"] = "left"}))},
      pandoc.Attr("", {"level-2-heading"}, {})
    )
  elseif level == 4 then
    -- 三级标题：仿宋_GB2312，三号，加粗，左对齐
    return pandoc.Div(
      pandoc.Para{pandoc.Span(header.content, pandoc.Attr("", {}, {["font-family"] = "仿宋_GB2312", ["font-size"] = "16pt", ["font-weight"] = "bold", ["text-align"] = "left"}))},
      pandoc.Attr("", {"level-3-heading"}, {})
    )
  end
  
  return header
end

function Para (para)
  -- 正文：仿宋_GB2312，三号，首行缩进2字符
  return pandoc.Div(
    para,
    pandoc.Attr("", {"body-text"}, {})
  )
end
```

4. 如有问题，调整模板中的样式设置

## 故障排除

### 常见问题

**Q: 字体显示为宋体而不是指定字体**
A: 
- 检查字体是否正确安装
- 在Word模板中重新设置样式
- 确保转换时使用了正确的模板文件

**Q: 标题没有居中**
A: 
- 检查Word模板中的段落格式设置
- 确认对齐方式设置为"居中"

**Q: 正文没有首行缩进**
A: 
- 在模板中修改正文样式的段落格式
- 设置"首行缩进"为2字符

**Q: 字体大小不正确**
A: 
- 检查模板中各样式的字号设置
- 二号=22pt，三号=16pt

## 质量检查清单

转换完成后，请检查：

- [ ] 公文标题：方正小标宋简体，二号，加粗，居中
- [ ] 一级标题：黑体，三号，加粗，左对齐
- [ ] 二级标题：楷体_GB2312，三号，加粗，左对齐
- [ ] 三级标题：仿宋_GB2312，三号，加粗，左对齐
- [ ] 正文：仿宋_GB2312，三号，首行缩进2字符
- [ ] 行距：1.5倍行距
- [ ] 页边距：符合公文标准（上3.7cm，下3.5cm，左2.8cm，右2.6cm）
